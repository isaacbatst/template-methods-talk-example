# Jogo de RPG

- Requisito 1:
  - Personagens tem:
    - Atributos: Força, Destreza e Inteligência (1 a 10)
    - Vida: 100
    - Podem atacar outro personagem, aplicando 10 de dano - destreza do alvo.

- Requisito 2:
  - Personagens podem ser de raças diferentes:
    - Elfo soma a inteligência + destreza no dano
    - Orc soma 2x a força no dano
